## 3. Usage
```bash
$ cd ~
$ git clone https://github.com/PINTO0309/OpenVINO-bin.git
$ cd OpenVINO-bin
```
### (1) Linux
```bash
$ Linux/download_2019R3.sh
```

### (2) RaspberryPi
```bash
$ RaspberryPi/download_2019R3.sh
$ echo "source /opt/intel/openvino/bin/setupvars.sh" >> ~/.bashrc
```
