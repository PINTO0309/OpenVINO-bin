## 3. Usage
```bash
$ cd ~
$ git clone https://github.com/PINTO0309/OpenVINO-bin.git
$ cd OpenVINO-bin
```
### (1) Linux
```bash
$ Linux/download_R5.sh
```

### (2) RaspberryPi
```bash
$ RaspberryPi/download_R5.sh
```
